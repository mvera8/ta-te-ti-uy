# next-auth-mantine-convex

## Stack
Next.js
Auth.js
Mantine Dev
Convex DB
Sileo
dnd-kit
Vercel

## Getting Started

First, run the development server:

```bash
npm run dev
```

To make changes in Convex DB:

```bash
npx convex dev
```
